# Student Registration System

A responsive application built for managing student records.
This system allows users to register new students, view existing records and modify records wihtout needing a backend database.

## Features


* **Data Handling**
    * Used browser's local storage to save all student information.
* **Validation**
    * All neccessary boxes are filled out.
    * Names use only letters.
    * Id's and contact numbers are numeric.
    * Contact number are atleast 10 digits long.
    * Email are formatted correctly using efficient regular expressions.   
* **Records Table**
    * Dynamic, easy-to-read, scrollable and designed with zebra-striping.
* **Control Over Records**
    * Having total control over data allowing creation, updation and deletion of records.


## Tech Stack

* **HTML**
    * Semantic structure for better accessibility.
* **CSS**
    * Styling, layout(Grid/Flexbox) and responsiveness.
* **JavaScript**
    * Validation and DOM manipulation.



# Installation


1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/Prathamesh910/Student_Registration_System.git]
    ```
2.  **Navigate to the Project Directory:**
    ```bash
    cd Student-Registration-System
    ```
3.  **Open the Application:** Locate the `index.html` file and open it directly in any modern web browser (Chrome, Firefox, Edge).
4.  **Register a Student:** Fill out the form fields in the "New Registration" section and click the **Submit** button.
